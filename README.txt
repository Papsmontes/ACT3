Modificaciones al juego paint.py

Se agregaron las siguientes funcionalidades

Primero importamos una libreria en la linea 16 y posteriormente hicimos una modificación para que fuera mas listo el fantasma, pintadolo de color amarillo camuflajendolo
y haciendo mas dificil el juego, entre las lineas 145 - 148.
Para cambiar el mapa se modifico entre la linea 30 - 51 añadiendo y eliminando secciones del mapa con los 0 y 1.
Para terminar hicimos mas rapido al fantasma modificando los vectores de la linea 134 - 137.



El archivo fuente donde se hicieron las modificaciones no es de nuestra propiedad, 
fue obtenido de las librerias de freegames de python, el archivo original es pacman.py
puede revisarse un poco de la libreria y de los archivos originales de los juegos en la siguiente URL:
http://www.grantjenks.com/docs/freegames/
